# Color Picker

A simple and minimalist Color Picker built with React.js

-Features
Responsive Design: This color picker is responsive, work on various screen sizes and devices.
User-Friendly: The minimalistic design provides a distraction-free experience focused on colors.

-[Live Application](https://color-picker-gules-beta.vercel.app/)

-Notes
During the development of this task, I encountered a challenge when initially fetching data from an API. After conducting a research and reading some documentaion about API, I successfully resolved the issue and implemented proper data fetching. This experience enhanced my problem-solving skills and deepened my understanding of API integration. Additionally, I took the opportunity to improve the user interface of the Color Picker by redesigning the layout and adopting a minimalist design approach.
